### [Store API: Django DRF](https://github.com/Stendal16471/store-api)
**Production-ready backend на Django REST Framework.** Завершённый учебный проект — REST API для каталога товаров с системой отзывов. Реализованы модели, сериализаторы, Function-Based и Class-Based Views, фильтрация с валидацией. Проект демонстрирует понимание чистой архитектуры Django (разделение на проект и приложение) и **корректные настройки для production** (`DEBUG=False`, безопасный `SECRET_KEY`, настроенный `ALLOWED_HOSTS`). API полностью работоспособно и возвращает данные в формате JSON.
> **Стек:** Python, Django, Django REST Framework, SQLite.
> **Статус:** **Проект завершён, протестирован и готов к демонстрации.**

### [Social Network API](https://github.com/Stendal16471/backend_for_social_network)
**Полноценный production-ready backend для социальной сети обмена фотографиями.** Проект демонстрирует промышленный подход: чистая архитектура Django REST Framework, поддержка множественных изображений, система лайков/дизлайков, комментарии, автоматическая геолокация через geopy. **Ключевая особенность:** 34 автоматических теста, покрывающих ключевую функциональность, и образцовая документация. Настройки приведены в соответствие с best practices (`DEBUG=False`, безопасное хранение секретов).
> **Стек:** Python, Django, Django REST Framework, PostgreSQL, geopy, Pillow.
> **Статус:** **Production-ready проект с полным тестовым покрытием.**

### [Car Shop: Django Website](https://github.com/Stendal16471/car-shop)
**Веб-сайт автосалона на Django.** Проект демонстрирует навыки работы с ORM, моделями и представлениями. Реализованы страницы списка автомобилей, детальной информации и статистики продаж, а также поиск по модели. Использованы готовые шаблоны и статические файлы. Настройки приведены в production-ready состояние.
> **Стек:** Python, Django, PostgreSQL.
> **Особенность:** Полноценный сайт, дополняющий API-проекты в портфолио.

### [Advanced SQL: Music Database](https://github.com/Stendal16471/Advanced-data-sampling)
**Проект, демонстрирующий продвинутые навыки работы с SQL на примере базы данных музыкальной библиотеки.** Включает полную схему БД (таблицы жанров, исполнителей, альбомов, треков, сборников с корректными связями многие-ко-многим), наполнение тестовыми данными и **более 20 сложных аналитических запросов** с использованием JOIN, подзапросов, агрегатных функций, GROUP BY, HAVING и текстового поиска.
> **Стек:** PostgreSQL, SQL.
> **Особенность:** Показывает понимание SQL на уровне, выходящем за рамки ORM.

### [Cat-as-a-Service: Yandex.Disk Uploader](https://github.com/Stendal16471/Coursework-API)
**Курсовая работа по интеграции с внешними API.** Python-скрипт, который генерирует изображение котика с произвольным текстом (через бесплатное API `cataas.com`) и загружает его на Яндекс.Диск через REST API. Проект демонстрирует уверенную работу с HTTP-запросами, обработку ответов, прогресс-бары и безопасное хранение токенов (вводятся в рантайме). Код чисто структурирован, ведётся история загрузок в JSON.
> **Стек:** Python, requests, tqdm, Yandex.Disk API.
> **Особенность:** Полноценный инструмент для автоматизации загрузки изображений в облако с понятным CLI-интерфейсом.
